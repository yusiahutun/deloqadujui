Have you finally finished playing Cyberpunk 2077's Phantom Liberty campaign, or are you just getting started? Either way you should check out some of the amazing mods from the community. These mods get you access to hours of additional content and playtime in the amazing sandbox that is Night City.

What to Know About Cyberpunk 2077 Modding

If you want to mod Cyberpunk 2077 there are a few modding tools that will make your job much easier. You can download most of the mods from Nexus Mods website and install them using a modding tool called Vortex Mod Manager. It will help you install and manage mods with relative ease. After that, you will also need to download different modding tools that are specific to Cyberpunk 2077 (depending on whether your mod needs them or not).

A majority of Cyberpunk 2077 mods require certain modding tools to work properly. Most of these tools (like Cyber Engine Tweaks, REDMod, redscript, and more) can be downloaded directly from Nexus Mods website and installed through Vortex Mod Manager. So, here's a brief explainer on some of the modding tools you will need to install.

  • Vortex Mod Manager: Vortex mod manager is an open-source mod manager that will allow you to download, install, and manage numerous mods. It is compatible with a vast variety of games and is compatible with Windows 7, 8, and 10, as well as Windows XP. It can also be used to install other modding tools.
  • Cyber Engine Tweaks: Cyber Engine Tweaks is a framework that allows modders to interact with the game's internal functions using Lua. It also provides a user-interface with a console and additional tools to assist developers. You can use the console to unlock weapons, vehicles, activate cheats, manipulate quest objectives, and much more.
  • REDMod: REDMod is the official modding tool for Cyberpunk 2077 and can be installed directly from Steam, Epic Games, and GOG. It is needed to install and deploy a vast variety of mods for the game.

If you plan on downloading a mod, go to its description and see if it has a requirement for the above-mentioned modding tools. If yes, download the modding tool using Vortex Mod Manager or manually install it to Cyberpunk 2077. More on that down below.

Where to Find and Install Cyberpunk 2077 Mods

We recommend that you check out the Nexus Mods website, where you can view, and download user created mods for Cyberpunk 2077. If you just want to install a mod that doesn't require any modding tools, then simply follow the instructions given in the mod description. Such mods just need to be extracted to the main Cyberpunk 2077 install folder or the mod folder. Here's what your folder path might look like:

D:\SteamLibrary\steamapps\common\Cyberpunk 2077\mods

There's also a vast majority of mods that need to deployed and activated through a modding tool or the Vortex Mod Manager. So, take a look down below on how to use these modding tools for Cyberpunk 2077.

Enabling Mods for Cyberpunk 2077

If you are playing Cyberpunk 2077 on Steam or Epic Games, make sure to turn on the "Enable Mods" option in the game's launcher (REDlauncher). This option can be enabled within REDlauncher by pressing the settings icon right beside the "Play" button.

If you are playing the game on GOG, then go to Cyberpunk 2077 > Settings > Manage Installation > Configure > Features. Scroll down and turn on the "Enable Mods" option. You will also need to change the "Default executable" to "Cyberpunk 2077 with mods".

How to Use Vortex to Install Mods for Cyberpunk 2077

Go to Nexus Mods and download the latest version of Vortex Mod Manager. We recommend that you install the mod manager to the same drive as Cyberpunk 2077 install folder. Once installed, open the mod manager and log in to your Nexus Mods account. (Register on the website if you don't have one.)

Once you're logged in, go to the "Games" tab and search for "Cyberpunk 2077". If the game doesn't show up in the app, then you will need to download the Cyberpunk 2077 Vortex Support extension by going to this link. Once downloaded, open Vortex > Extensions > Drop File > Select the downloaded zip file.

Restart the Vortex Mod Manager and Cyberpunk 2077 will now show up in the Games section. ​​​​​​Click on "Manage" to add the game.

The mod manager automatically detects the game files but if it doesn't, it will ask you for its install location. Go to its install folder and select the Cyberpunk 2077 folder. If you don't know the game's install location, then follow the steps below to find it.

Steam: Steam Library > Right-click on Cyberpunk 2077 > Manage > Browse Local Files

GOG: GOG Library > Right-click on Cyberpunk 2077 > Manage Installation > Show Folder

Epic Games: Library > Cyberpunk 2077 > Three-dot menu > Manage > Press the "file" icon in front of Installation.

Once the game has been located, enable hardlink deployment by going to Settings > Mods > Deployment Method. Now, you can install mods directly through Vortex.

Go to Nexus Mods website and download any Cyberpunk 2077 mod using the Vortex Mod Manager option. You can download the mod file as a zip folder by clicking on the "Manual" download button or directly through the Vortex mod manager by clicking on the "Vortex" download option.

If you choose the Vortex option the website will automatically open Vortex and install the mod directly into the game. In Vortex, go to the Mods section to see which mods are enabled.

You can now install mods for Cyberpunk 2077 without any extra work. Plus, if a mod requires additional modding tools you can download them directly from Nexus Mods and install them easily using the Vortex Mod Manager.

How to Use Cheats in Cyberpunk 2077 Using Cyber Engine Tweaks Mod

Cyber Engine Tweaks (CET) is a necessary application for a lot of Cyberpunk 2077 mods. It also comes with a command console that can be used to unlock various weapons, vehicles, and much more. You can download Cyber Engine Tweaks from Nexus Mods either manually or through Vortex Mod Manager. If you have downloaded it manually, unzip the file and drag the bin folder to your Cyberpunk 2077 game directory.

When you launch the game for the first time after installing the mod, you will get a "CET First Time Setup" prompt. Click the "Not Bound" button and set a key bind to enable/disable the mod overlay. We recommend using the tilde (~) or backslash (\) key since they aren't commonly reserved for other purposes.

Once you have set the key bind, you can bring up the overlay by pressing that key. You can now use all kinds of cheats using the CET console. These cheat commands can be downloaded from Nexus Mods or copied from this webpage.

After copying any command, go to the console section of the mod and paste it. Press "Enter" to activate the command and the cheat will now be activated in the game.

Don't worry, you won't be banned for using mods and cheats in Cyberpunk 2077, as it is a single player game. So, go ahead and use these mods however you want.

How to install REDMod for Cyberpunk 2077

REDMod is the official modding tool developed by CD Projekt Red for Cyberpunk 2077. Many mods created by the community require the REDMod tool to function properly. Fortunately, installing the mod is quite easy as it is listed as a free DLC for Cyberpunk 2077 on Steam, Epic Games, and GOG.

  • REDMod DLC for Steam
  • REDMod DLC for GOG
  • REDMod DLC for Epic Games (pre-installed with Cyberpunk 2077)

Once you have installed REDMod to your game, you can use any mod that requires the aforementioned modding tool as a requirement. Now, all you need to do is download and install any mod manually or by using the Vortex Mod Manager. You can finally play with hundreds of community created mods for Cyberpunk 2077. Enjoy.
